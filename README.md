# DXF 房间面积提取

从 CAD 导出的 DXF 变电站图纸中自动提取房间净面积。

## 方法

**墙线内面法 (wall_bounds)** — 从房间文字标注位置向四方向找到最近的墙内面作为边界。

**三源交叉验证：**
- 图框标注总建筑面积（粗校）
- 轴线网格推算外包面积（几何基准）
- 房间明细表逐室核对（如有）

## 主要功能

- 双线墙识别（间距 140-280mm）
- 自动扣除楼梯间、风井
- L形房间处理（中心点+多点采样）
- 走廊区域识别（全高走廊、T字形走廊）
- 散热平台单独计算
- 偏差分级处置（<3% ✅ / 3-10% ⚠️ / >10% ❌）
- 输出审计留痕（每个房间可追溯到围合墙线）

## 依赖

```bash
pip install ezdxf shapely matplotlib openpyxl
```

## 使用

修改 `dxf_path` 为你的 DXF 文件路径，运行：

```bash
python final_v16_reference.py
```

## 输出

- 控制台输出：每个房间面积、总面积、与建筑面积对比
- 可视化：房间着色图（保存为 PNG）
- 可选：XLSX 输出（含房间编号、面积、围合墙线坐标）

## 适用范围

- 建筑/电力工程 DXF 文件（AutoCAD 2010+）
- 双线墙表示的房间边界
- 房间名称在 TEXT 实体中（标注层或 PUB_TEXT 层）

## 已知限制

- 不接受单线墙/中心线 DXF
- 不接受 IFC/BIM 格式（需先导出为 DXF）
- 不处理曲面墙体
