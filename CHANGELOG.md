# Changelog

## v2.3.0 (2026-06-18)

### 新增
- **墙线缺口延伸法**: 墙覆盖率<80%且单段缺口>2m时，仅缺口处延伸(不穿越完整墙段)
- **110kV变电站实战验证**: 地下一层(B1)、一层(1F)、二层(2F)完整面积计算
- **上空不计建筑面积**: "上空"房间(中庭/大厅)按GB/T 50353不重复计算
- **同名房间连通合并**: 同名房间仅在连通(无墙分隔)时合并，不连通分开列
- **色块图zorder规则**: 墙线/轴网画在色块上面(zorder=20, alpha=0.9)
